Что это?

Это номеронабиратель (далее: НН), специальная приставка к городскому телефону или GSM-шлюзу.

Нужен чтобы организовать "быстрый набор" номеров из 100 запрограммированных ячеек.

Как выглядит

Спереди:

  • цифровая клавиатура с кнопками подтверждения и отмены
  • два светодиодных индикатора 8x8
  • наше фото :)

С верхнего торца:

  • разъём питания USB 5V 1A
  • разъём RS232 для подключения dialup модема
  • тумблер режима работы: для настройки направлен к краю, для работы - к центру
  • кнопка перезагрузки

С левого края:

  • технологический тумблер для программирования, в рабочем режиме направлен вниз

Что умеет

  • режим "wifi": настройка телефонной книги и параметров
  • режим "работа": вызов номеров из сохранённых ячеек с помощью подключенного модема

Смена режима выполняется с помощью тумблера и кнопки перезагрузки.

Подготовка к работе

  • подключить внешний модем к телефонной линии: разъём LINE (ближе к центру)
  • подключить телефон ко внешнему модему: разъём PHONE (ближе к краю)
  • подключить НН ко внешнему модему: кабель RS232
  • подключить питание к модему: внимание, для Acorp 56000 адаптер 9V ~AC~ (в комплекте)
  • подождать, пока индикаторы модема стабилизируются (5-10 секунд)
  • подключить питание к НН: USB 5V 1A

Выход на режим

При выходе на режим, устройство делает:

  • тест контроллера клавиатуры (возможен код ошибки 1)
  • тест модема (остальные коды ошибки)
  • после чего пищит и отображает либо мигающую точку (работа), либо надпись Wifi (настройка)

При отображении кода ошибки клавиатуры, должна помочь кнопка перезагрузки.

При отображении кода ошибки модема, можно пробовать исправить ситуацию "на ходу":

  • если не соединён, подключить кабель модема
  • выключить и включить питание модема

Режим "Wifi"

В этом режиме НН создаёт точку доступа Wifi, к которой можно подключиться с помощью
любого устройства (ПК, телефон, планшет).

SSID точки доступа NaborNomera
Пароль для простоты отсутствует (в основном режиме Wifi выключен).

Обычно, после соединения страница настроек открывается автоматически - но если этого не происходит,
нужно открыть браузер и перейти на адрес http://nn.local

Должна открыться страница настроек устройства. Разделы страницы будут описаны ниже:

Справка

Текст, который вы читаете, доступен с самого устройства по этой ссылке.

Отладка порта RS232

Оставлена как возможность. Поле для ввода поддерживает AT-команды или, в случае ввода цифры,
осуществляет набор номера из заданной ячейки

Добавить запись

  • ячейка: номер от 0 до 99
  • описание: 25 практически любых текстовых символов (цифры, русские или английские буквы, знаки)
  • тип номера: в случае мобильного, модем ждёт второго гудка (см. далее: префикс набора). Городской будет набран как есть.
  • номер: до 28 цифр. В случае мобильного, вводить нужно начиная с кода региона (т.е. для 8-921... восьмёрку вводить НЕ нужно)
  • кнопка Записать: запись в память. Может выдать подсказку по заполнению полей.

Настройки

  • префикс выхода на мобильную связь: до 10 символов. Типичное значение 8W, это значит набрать 8 и ждать гудка. Возможны варианты 8, (значит ждать без подтверждения).
  • яркость, звук нажатия клавиш: в случае сохранения, применяются при перезагрузке
  • метод набора номера: применяется в рабочем режиме
  • кнопка Записать: запись в память. Может выдать подсказку по заполнению полей.

Телефонная книга

Таблица, где первая колонка - индекс ячейки (0-99), за которой следуют номер и описание.

Режим "Работа"

В этом режиме Wifi отключен, НН отображает мигающий курсор, ожидает ввода номера ячейки.

Чтобы осуществить вызов, необходимо:

  • ввести номер ячейки (для первого десятка поддерживаются любые варианты, например: 7, 07)
  • набор можно отменить кнопкой "Отмена" или через 30 секунд бездействия
  • для вызова, нажать кнопку подтверждения
  • в случае ошибки связи с модемом, будет высвечен код. Рекомендуется перезагрузка.
  • в случае удачной отсылки команды, номер ячейки замигает
  • во время набора, нажатие кнопки НН отменяет вызов
  • после набора, НН некоторое время (порядка 10 с) "держит трубку". Нужно успеть поднять трубку на телефонном аппарате, чтобы вызов не был сброшен.

Коды ошибок

2Не отвечает модем
3Модем: нет несущей
4Модем: нет гудка
5Модем: занято
6Модем: прочие ошибки
